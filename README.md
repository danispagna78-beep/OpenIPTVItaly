# 🇮🇹 OpenIPTVItaly

**OpenIPTVItaly** è una **playlist IPTV italiana open-source, pubblica e curata**, progettata per diventare **la playlist di riferimento per i canali televisivi italiani in chiaro**.

🎯 Obiettivo: qualità, affidabilità e trasparenza. Nessun link non funzionante, nessuna pay-TV, nessun DRM.

## 📺 Contenuto

La playlist include **solo flussi disponibili pubblicamente e legalmente accessibili**:

* 📡 Canali TV nazionali e locali gratuiti
* 🧒 Canali per bambini
* 📰 Notizie
* 🎬 Intrattenimento
* 🏟️ Sport gratuito
* 📻 Radio (quando disponibili)

❌ **Non sono inclusi:**:

* Canali TV a pagamento
* Streaming protetto da DRM
* Contenuti pirata o ridistribuiti

## 🧩 Playlist Structure

The playlist follows common IPTV standards:

* Consistent `tvg-id` for EPG matching
* Official `tvg-name`
* High-quality `tvg-logo`
* `tvg-chno` (LCN where applicable)

## 📡 EPG Source

When available, EPG data is obtained from publicly accessible sources provided by **epgshare01.online**.

The EPG is used for informational purposes only and reflects standard TV scheduling
data that is already publicly available.
